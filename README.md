# Proyecto-QA-Diseno-y-funcionalidad-de-Urban-Routes
Se probó la funcionalidad de compartir un automóvil en 2 entornos: Google Chrome 800x600 y Firefox 1920x1080

# LOGROS
1. Creé una lista de comprobación para el diseño del formulario de reserva
2. Creé una lista de comprobación para la funcionalidad de las ventanas del "Método de pago" y "Agregar tarjeta"
3. Preparé casos de prueba para el botón "Reservar"
4. Preparé casos de prueba para la función de reservar automóviles
5. Probé los casos de la lista de comprobación para el diseño del formurmulario de reserva en los navegadores con resolución:

   - Google Chrome 800x600
   - Firefox 1920x1080
6. Encontré bugs y comportamientos inesperados, los cuales documente en un informe de errores creado en Jira
